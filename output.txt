Main learning concepts
. Computational complexity The study of the resources time, space, etc. required to solve computational problems.
. Time complexity The measure of the amount of time taken by an algorithm to solve a problem as a function of the input size.
. Space complexity The measure of the amount of memory required by an algorithm to solve a problem as a function of the input size.
. Worstcase complexity The maximum amount of resources required by an algorithm to solve a problem for any input of a given size.
. Halting problem The problem of determining whether a program will terminate or continue to run indefinitely.
